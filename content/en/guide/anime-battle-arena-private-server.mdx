<Section icon={Target} title="Step-by-Step: Setting Up Custom Game Modes">

Configuring an **anime battle arena private server** for unique game modes requires a systematic approach. Follow these steps to host a seamless custom match for your community.

<Steps>
  <Step num="1" title="Acquire and Launch Your Server">
    Purchase or join a private server link via the Roblox game page. Ensure all participants are on your friends list or have the direct join link.
  </Step>
  <Step num="2" title="Define the Match Theme">
    Select a theme such as Clone Summons, Bald Characters Only, or Giants vs Projectile Spammers. Restricting character selection makes matches highly entertaining.
  </Step>
  <Step num="3" title="Configure Lobby Settings">
    Set the health pools, select a balanced map (avoiding universally disliked maps like Chimera Ant Map), and determine if friendly fire should be toggled on.
  </Step>
  <Step num="4" title="Execute and Moderate">
    Start the match. Keep an eye on player behavior and lag spikes. Use server commands to reset or adjust parameters if a character bugs out.
  </Step>
</Steps>

<Callout type="warning" title="Character Bugs">
