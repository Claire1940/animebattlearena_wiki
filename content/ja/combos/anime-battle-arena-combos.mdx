<Section icon={Map} title="基本メカニズムと操作レイアウト">

**Anime Battle Arenaのコンボ**を理解するには、まず入力システムをマスターする必要があります。2026年の環境でも、メタは「M1（左クリック）」の効率性に焦点を当てています。通常攻撃によるヒットストン（硬直）を作り出し、そこからスキルや空中打ち上げへ繋げます。最初のM1ストリングを外すと反撃コンボを受ける隙ができるため、攻撃だけでなく間合い管理や移動も重要です。

Dogs Studios: Southが提供する操作系は、すべてのハイレベルな戦いの基礎となります。キャラ固有のヒットボックスに最適な間合いを維持するために、ダッシュとガードを自在に切り替える必要があります。

| 入力 | アクション | コンボでの戦略的用途 |
| :--- | :--- | :--- |
| **Mouse1** | 通常攻撃 | コンボの起点およびヒットストン生成 |
| **Space + Mouse1** | 空中攻撃 | 敵を打ち上げてラダーに繋ぐための「アッパー」 |
| **Q** | 回避 | 敵のコンボストリングから脱出するために必須 |
| **F** | ガード | コンボの始動を防ぎ、反撃のチャンスを作る |
| **1, 2, 3, 4** | スペシャルスキル | ストリングを延長したり、トドメを刺すためのダメージ源 |
| **Shift** | ダッシュ | 間合いを制御し、敵の攻撃を誘い出す |
| **G** | 変身 | ステータスを大幅強化し、通常はクールダウンをリセットする |

<Callout type="tip" title="プロの間合い管理">
Shiftキーを使って、相手のM1の範囲外で「ダッシュダンス」を行いましょう。相手が攻撃を空振りして硬直状態になった瞬間が、自分のコンボを開始する絶好のチャンスです。
</Callout>

</Section>

<Section icon={Target} title="コンボ構築の基礎">

2026年のPvPで勝つには、単なるボタン連打以上の技術が必要です。ほとんどの**Anime Battle Arenaコンボ**は「始動・延長・フィニッシュ」という特定の構造に従っています。最も一般的な延長手段は「ラダー」で、空中攻撃で相手を打ち上げ、追いかけてM1ストリングを継続します。

<Steps>
  <Step num="1" title="M1オープナー">
    地上でM1を3〜4回ヒットさせます。これにより相手はヒットストン状態となり、即座にガードや回避ができなくなります。打ち上げる予定がある場合は、最後の5段目まで出し切らないように注意してください（最後の一撃は通常ノックバックを伴うため）。
  </Step>
  <Step num="2" title="アッパー（打ち上げ）">
    相手が硬直している間に、SpaceとMouse1を同時に押します。これで敵を空中に打ち上げます。これが高ダメージコンボへの重要な分岐点です。
  </Step>
  <Step num="3" title="空中延長">
    打ち上げ直後にジャンプし、空中でM1ストリングを継続します。キャラの重力や攻撃速度にもよりますが、通常は相手が落下するまでにさらに3〜4ヒットさせることが可能です。
  </Step>
  <Step num="4" title="スキルの組み込み">
    相手が落下し始めるタイミング、または空中ストリングの終わりにスキル（1-4）を使用します。相手を地面に叩きつけるスキルや、再度コンボへ引き戻すスキルを活用しましょう。
  </Step>
</Steps>

<Callout type="warning" title="回避の管理">
キルが100%確定している状況以外で、回避（Q）を攻撃的に使うのは避けましょう。無駄な回避は、相手のフルコンボに対して完全に無防備な状態をさらすことになります。
</Callout>

</Section>

<Section icon={Shield} title="競技PvPの戦術と防御">

ランク戦（1v1や2v2）では、**Anime Battle Arenaのコンボ**ができるだけでは半分しか勝てません。相手にコンボをさせない方法を知る必要があります。ABAの防御は「ニュートラルゲーム（両者がコンボ中ではない状態）」を中心に構築されています。

| 防御戦術 | 最適な使用タイミング | リスクレベル |
| :--- | :--- | :--- |
| **ガード (F)** | 相手が予測可能なM1ストリングを使っている時 | 低（ガードブレイクされる可能性あり） |
| **回避 (Q)** | 相手が高ダメージのスキルを使おうとしている時 | 中（長いクールダウン） |
| **間合い管理** | 相手が攻撃的でM1を狙っている時 | 低（優れた移動技術が必要） |
| **カウンター** | 相手が隙の大きい技を終了した時 | 高（完璧なタイミングが必要） |

勝利数を増やすには「反撃」への集中が不可欠です。「ガードブレイク」や大きなフィニッシュ技を空振りした相手は、アニメーションの硬直に囚われます。そこが最もダメージを与えられるコンボの始動タイミングです。

<Callout type="info" title="サーバーパフォーマンス">
2026年現在、サーバーの遅延がコンボのタイミングに影響を与えることがあります。「ピンのスパイク（ラグ）」を感じる場合は、コンボを簡略化しましょう。複雑な空中ラダーを避け、信頼性の高い地上M1ストリングから確実にスキルへ繋ぐのが賢明です。
</Callout>

</Section>
