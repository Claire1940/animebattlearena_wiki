<Section icon={Swords} title="基本技セット：ROOM外とROOM内">

ローは、ROOMの中に立っているかどうかによって2つの異なる技セットを持ちます。これら2つの状態を管理することが、ハイレベルなプレイの鍵となります。

### ROOM外の技

ドメインが展開されていない時、ローは素早く直接的な物理攻撃を頼りに、相手の隙を突いたり身を守ったりします。

| 技名 | ダメージ | クールタイム | 特性・用途 |
| :--- | :--- | :--- | :--- |
| **斬撃（Slash）** | 20 | 14秒 | ガード崩し、コンボ始動、近接 |
| **カウンターショック** | 22 | 8秒 | ガード崩し、ノックバック、飛び道具 |
| **ROOM** | N/A | 5秒 | 技セット変更、初期サイズ12スタッド |

### ROOM内の技

**ROOM**が有効になると、ローの標準スキルは強力な空間操作能力へと変化します。

| 技名 | ダメージ | クールタイム | 特性・用途 |
| :--- | :--- | :--- | :--- |
| **シャンブルズ** | N/A | 7秒 | テレポート、入れ替え、移動 |
| **切断（Amputate）** | 28 | 33秒 | 多段ヒット、ガード崩し（2打目）、コンボ延長 |
| **切断・スマッシュ** | 63 (合計) | 33秒 | 2体同時叩きつけ、ブロック無視 |
| **タクト（Takt）** | 21-42 | 3秒/10秒 | 岩の召喚/投擲、叩きつけ、召喚時無敵判定 |
| **ガンマナイフ** | 32 | 16秒 | ガード崩し、ノックバック、コンボフィニッシュ |

<CardGrid cols={3}>
  <Card title="シャンブルズの極意">
    シャンブルズを防御的に使って味方を救ったり、攻撃的に使ってコンボの最中に相手と位置を入れ替えたりしましょう。
  </Card>
  <Card title="タクトによる防御">
    タクトで岩を召喚すると、短い無敵時間（I-frames）が発生します。また、岩は物理的な地形として機能し、飛んでくる飛び道具を遮断できます。
  </Card>
  <Card title="切断・スマッシュ">
    切断の2打目で別の相手にヒットさせると、ローは両方のターゲットを一緒に叩きつけ、凄まじいダメージを与えます。
  </Card>
</CardGrid>

<Callout type="warning" title="ROOMの縮小">
通常のROOMは10秒ごとに1.5スタッドの割合で徐々に縮小し、最小3x3スタッドまで小さくなることを忘れないでください。戦闘中に強化スキルを失わないよう、境界線を常に把握しておきましょう。
</Callout>

</Section>

<Section icon={Sparkles} title="オペオペの実の真骨頂：覚醒ガイド">

変身技**「死の外科医」**を発動することで、ローの真のポテンシャルが解放されます。ROOMは34x34スタッドの巨大なサイズに拡大し、縮小しなくなるだけでなく、常にローを中心に追従するようになります。

### 覚醒技セット

| 技名 | ダメージ | クールタイム | 特性・主要な用途 |
| :--- | :--- | :--- | :--- |
| **タクト（吹き飛ばし）** | 31 | 1秒/14秒 | 相手を飛ばす、境界で自動叩きつけ |
| **タクト（岩石粉砕）**| 30 | 2秒/14秒 | 巨大な岩の召喚・投擲、コンボ始動 |
| **シャンブルズ** | N/A | 9秒 | ブロック無視のテレポートと入れ替え |
| **ラジオナイフ** | 35 | 18秒 | ガード崩し、範囲攻撃、多段ヒット、打ち上げ |
| **メス** | 即死 | N/A | HP20以下のターゲットを即座に処刑 |

<Steps>
  <Step num="1" title="覚醒の発動">
    モードを発動して、即座に巨大な移動式ROOMを展開します。これにより相手の逃走を防ぎます。
  </Step>
  <Step num="2" title="ターゲットの孤立">
    タクトを使用して脅威となるターゲットを味方から引き離し、境界での叩きつけを利用して追撃の準備をします。
  </Step>
  <Step num="3" title="処刑の確定">
    ラジオナイフと基本攻撃で圧力をかけ、相手のHPが20以下になったら「メス」を使用して即座に仕留めます。
  </Step>
</Steps>

<Callout type="success" title="メスの処刑ルール">
他の処刑パッシブとは異なり、ローの「メス」は技として手動で発動する必要があります。相手の最大HPの割合に関係なく、HPが厳密に20を下回っている場合にのみ機能します。
</Callout>

</Section>

<Section icon={Target} title="高度なコンボシーケンス">

通常状態と覚醒状態の両方でローのダメージ出力を最大化するには、正確なコンボの実行が不可欠です。

### 通常時のコンボ

- **コンボ 1（簡易ガード崩し）:** 通常攻撃(M1) 5回 + **カウンターショック**
- **コンボ 2（ROOM延長コンボ）:** M1 4回 + **切断** + M1 4回 + **ガンマナイフ**
- **コンボ 3（タクトラダー）:** （タクト発動中）M1 3回 + 空中打ち上げ（Space + M1） + **切断** + ラダースラム + **タクト**

### 覚醒時のコンボ

- **コンボ 4（基本吹き飛ばし）:** M1 4回 + Space（地上） + **タクト**
- **コンボ 5（アルティメット処刑）:** （メス使用可能時）M1 3回 + 空中打ち上げ（Space + M1） + **メス** + ラダースラム + **ラジオナイフ** + **タクト**

| コンボ名 | 難易度 | 推定ダメージ | 主な用途 |
| :--- | :--- | :--- | :--- |
| **コンボ 1** | 初級 | 低〜中 | 素早いダメージ / ガード崩し |
| **コンボ 2** | 中級 | 高 | 標準的なROOM内戦闘 |
| **コンボ 3** | 上級 | 非常に高 | 高度な空間制御セットアップ |
| **コンボ 4** | 初級 | 中 | 素早い位置取り / 吹き飛ばし |
| **コンボ 5** | 上級 | 致命的 | 低HPの相手を確実に仕留める |

<Callout type="tip" title="コンボの最適化">
2v2や3v3の試合では、アスナやオールマイトなどの味方と連携しましょう。アスナの「スイッチ」は、ローの「切断」でスタンした敵に対して容易に追撃でき、回避不能なワンショット（即死）状況を作り出せます。
</Callout>

</Section>
