<Section icon={Shield} title="通常形態：タクティカル・エスピオナージ・アクション">

通常形態の**Anime Battle Arena ソリッド・スネーク**は、ユーティリティと隠密を駆使してニュートラルゲームをコントロールします。彼のキットは、周囲を無視して過度に攻撃的なプレイヤーを罰するように設計されています。

**Mk22 麻酔銃**
Mk22はスネークの基本となる飛び道具です。10秒のリロードクールダウンの前に7発の弾丸があります。単一のターゲットに4発命中させると、相手は10秒間「崩れ落ちる」ような睡眠状態になります。これはあらゆるコンボやリセットのための自由なチャンスとなります。

**ダンボール箱＆チャフグレネード**
ダンボール箱は、敵の黄色い視界コーンの外にいる限り透明化を提供します。さらに重要なのは、これが「身代わり」として機能し、スネークがダメージを受けることなく1回分の攻撃を耐えることができる点です。チャフグレネードはガードを崩し、敵のクールダウンを凍結させるフィールドを作り出す多機能なツールです。

| 技名 | 主な用途 | 特殊効果 |
| :--- | :--- | :--- |
| **Mk22** | コンボ延長 / CC | 4発命中 = 10秒間の睡眠 |
| **ダンボール箱** | 防御 / 隠密 | 1ヒット無効化 |
| **チャフグレネード** | ガード崩し / 妨害 | 敵のクールダウンを停止 |
| **ホールドアップ** | ダメージバフ | 次のMk22/M1ダメージが2倍 |

<Callout type="warning" title="重要">
ダンボール箱を使用している間は、自分自身のクールダウンも停止します。Mk22やチャフグレネードの回復を待っている場合は、箱の中に長く留まりすぎないようにしてください。位置取りの変更や、致命的な究極技を回避するために短時間使用しましょう。
</Callout>

**ホールドアップの習得**
ホールドアップはおそらく最も当てるのが難しい技ですが、その分見返りも最大です。相手の真後ろに位置する必要があります。成功すると、相手は5秒間スタン状態になります。次のM1またはMk22のショットは2倍のダメージを与えます。2026年のメタでは、チャフグレネードのスタンを利用して相手の背後に回り込み、簡単にホールドアップを決めるプレイヤーが多く見られます。

</Section>

<Section icon={Sparkles} title="アラートフェーズ：武力行使許可">

スネークが覚醒を発動すると、**アラートフェーズ（警戒フェーズ）**に入ります。移行は無線通信（コーデック）から始まり、スネークが膝をつく短いアニメーションがあります。アラームが鳴り響き、画面が赤く点滅すると、スネークは迅速な排除のために設計された致命的な兵器へと切り替えます。

**M4＆ステルス迷彩**
M4はMk22に取って代わり、致命的な弾丸を高速連射します。多少の拡散はありますが、近中距離で命中させれば、相手の体力バーの半分を簡単に削り取ることができます。ステルス迷彩はダンボール箱の直接的なアップグレード版で、視界コーンの制限なく完全な透明化を提供します。攻撃を受けている最中に使用すると、回避行動としてのバックステップ（バク転）として機能します。

**手榴弾＆首絞め**
手榴弾（HE Grenade）は放物線を描いて飛び、大規模なガード崩し性能を持つ爆発を引き起こします。壁に跳ね返る性質があるため、軌道が予測しにくいのが特徴です。首絞め（Chokehold）はガードを無視するコマンド投げです。ボタンを連打することで、最大12回まで相手を絞めることができ、最終的に2秒間のダウンスタンを誘発します。

| 覚醒技 | ダメージポテンシャル | 主要メカニクス |
| :--- | :--- | :--- |
| **M4** | 非常に高い | ノックバックを伴うバースト射撃 |
| **ステルス迷彩** | N/A | 透明化 + 回避バックステップ |
| **手榴弾** | 40 (直撃) | ガード崩し / 4回跳ね返る |
| **首絞め** | 36 (合計) | ガード無視 / 2秒スタン |

<Callout type="success" title="勝利戦略">
アラートフェーズ中は、ステルス迷彩を回避ツールとして使うことに集中しましょう。相手のコンボに捕まったら、ステルス迷彩で離脱し、相手が攻撃アニメーションから回復する前に即座にM4バーストで反撃してください。
</Callout>

</Section>

<Section icon={Target} title="ステップバイステップ：コンボマスター">

**Anime Battle Arenaのソリッド・スネーク**で圧倒するには、彼特有のコンボタイミングをマスターする必要があります。M1が3段しかないため、100から0まで削り切るには延長技が不可欠です。

<Steps>
  <Step num="1" title="基本の「パンとバター」">
    M1を1回当て、続けて上強攻撃（Up-tilt）を繰り出します。相手が空中にいる間にMk22を1発撃ちます。相手が落下してきたら下叩きつけ（Down-slam）を決めます。これで追撃に最適な距離が作れます。
  </Step>
  <Step num="2" title="ホールドアップループ">
    ステップ1の下叩きつけの直後に、ホールドアップを使用します。相手は受け身の最中なので、簡単に背後を取ることができます。これに続けてM1を3回当て、さらにMk22を撃つことで、2倍ダメージバフを最大限に活用します。
  </Step>
  <Step num="3" title="睡眠フィニッシャー">
    試合中にすでにMk22を2発当てている場合、このコンボで睡眠状態が発動します。相手が眠ったら、自分のクールダウンを待つか、フルM1コンボからチャフグレネードを繋ぎ、目覚めた瞬間に確定でガードを崩しましょう。
  </Step>
  <Step num="4" title="覚醒首絞めチェイン">
    アラートフェーズでは、M1を2回当てた直後に首絞めを繰り出します。ボタンを連打してフルで12回絞めます。相手が倒れ込むところに、至近距離からM4バーストを叩き込みます。このシーケンスは、回避技なしで逃げることはほぼ不可能です。
  </Step>
</Steps>

<Callout type="tip" title="コンボ効率">
スネークの「コンボ1」（M1 + 上強 + Mk22 + 下叩きつけ + ホールドアップ + 3段M1 + Mk22）は約39.8ダメージを与えます。2026年のバランス調整では、これに加えてもう一度フルM1コンボを決めれば、撃破圏内に入ります。
</Callout>

</Section>
