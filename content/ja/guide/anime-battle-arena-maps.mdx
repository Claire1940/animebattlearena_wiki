<Section icon={Swords} title="アリーナの類型とキャラクターの相性">

Anime Battle Arenaの各アリーナは、特定のタイプに分類されます。これらのタイプを認識することで、ロード画面中にプレイスタイルを調整できます。**ブロリー（Legacy）**や**五条悟**のようなトップティアのキャラクターは、ほとんどのマップで通用するツールを持っていますが、状況に応じたピックは地形がキットに合っていないと苦戦する可能性があります。

  <Card title="狭い / 閉鎖的なアリーナ">

    - **主要戦略：** 積極的なM1コンボと素早いガードブレイク。
    - **リスク：** 変身（G）に巻き込まれる可能性が高い。
  </Card>
  <Card title="広い / 開放的なアリーナ">
    - **最適：** ゾナーや機動力重視のキャラクター。
    - **主要戦略：** 間合いの管理と、相手の空振りを罰することに集中。
    - **リスク：** 長距離アルティメットがないとトドメを刺しにくい。
  </Card>
  <Card title="垂直 / 複雑なアリーナ">
    - **最適：** 空中追尾スキルを持つキャラクター。

キャラクターを選択する際は、そのキャラクターの「覚醒（Awakening）」や変身がマップサイズとどう連動するかを考慮してください。広範囲（AoE）のアルティメットを持つキャラクターは、狭いマップでは壊滅的な威力を発揮しますが、ナメック星やトレーニングルームのような広いマップでは簡単に回避される可能性があります。

| キャラクターランク | 最適なマップタイプ | 理由 |
| :--- | :--- | :--- |
| **Sティア (例: クリリン)** | 全般 / 中規模 | 安定した圧力と速い飛び道具がどの規模にも適合 |
| **Aティア (例: 藍染)** | 広い / 開放的 | リワークされたキットにより優れた間合い管理とカイトが可能 |
| **Bティア (例: イタチ)** | 中規模 / 閉鎖的 | 幻術系の技は狭い場所ほど回避が困難 |
| **Sティア (例: ビッグ・マム)** | 狭い | 圧倒的なAoEの圧力により、敵の逃げ場をなくす |

<Callout type="warning" title="マッチアップ注意">
機動力の低いキャラクターで、広いマップの強力なゾナーと戦う場合、闇雲に追いかけないでください。相手が特殊技（1-4）を使い切るのを待ってから、Q回避を使って距離を詰めましょう。
</Callout>

<Section icon={Zap} title="異なる地形での移動をマスターする">

**Anime Battle Arenaのマップ**を攻略するには、ゲームの移動メカニクスを深く理解する必要があります。どのように動くかによって、自分がコンボを開始できるか、あるいはガード硬直に陥るかが決まります。

<Steps>
  <Step num="1" title="スプリントと間合いの管理">
    **Shift**キーを使用して、常にスプリント状態を維持してください。これにより、相手のM1射程のすぐ外側に留まることができます。広いマップでは、Q回避を浪費せずに長距離の飛び道具を避けるためにスプリントが不可欠です。
  </Step>
  <Step num="2" title="Q回避の極意">
    **Q**キーは最も価値のある防御ツールです。障害物が多いマップでは、回避を使って小さなオブジェクトを通り抜けたり、特殊技を繰り出した直後の相手の背後に回り込んだりしましょう。
  </Step>
  <Step num="3" title="空中機動">
    **Space + Mouse1**を組み合わせることで空中攻撃が可能です。これを使って相手を打ち上げます。高低差のあるマップでは、空中攻撃によって攻勢を維持したまま低地から高地へ移動することができます。
  </Step>
  <Step num="4" title="ブロック(F)の活用">
    コーナーや壁に追い詰められたときは、**F**を長押ししてブロックしてください。ブロック中は相手のリズムを観察しましょう。多くのプレイヤーはM1攻撃を3回当てた後にガードブレイク技を使おうとします。それが回避やカウンター攻撃の合図です。
  </Step>
</Steps>

| 入力コマンド | アクション | マップ戦略の文脈 |
| :--- | :--- | :--- |
| **Shift** | スプリント | 広いアリーナを素早く横断するために不可欠 |
| **Q** | 回避 | コーナーからの脱出やAoEアルティメットの回避に使用 |
| **F** | ブロック | マップの境界線に追い詰められた時に最適 |
| **Space + M1** | 空中攻撃 | 地上のトラップから離れてコンボを延長するために使用 |
| **G** | 変身 | 最大のリーチを確保するため、マップの中央で使用するのがベスト |

<Callout type="success" title="練習のヒント">
プライベートサーバーを使って「壁ジャンプ」やエアダッシュを練習しましょう。管理された環境で移動をマスターすれば、ランクマッチの1v1で非常に捉えにくい存在になれます。
</Callout>

<Section icon={Target} title="戦略的なポジショニングとマップコントロール">

ハイレベルなAnime Battle Arenaの対戦では、マップの「中央」を制圧したプレイヤーが勝利することが多いです。中央を維持することで、移動の選択肢が最大限に広がり、簡単にコーナーへ追い詰められるのを防ぐことができます。

**主要なポジショニング戦術：**
- **センターコントロール：** 常に中央付近に位置取り、360度どの方向にも逃げられるようにします。
- **壁ハメ：** 攻撃を当てたら、相手を最寄りの壁に向かって吹き飛ばすようにしましょう。これにより、相手の横回避の能力が制限されます。
- **ベイト（誘い）：** トラップや崖の近くに立ち、相手にリスクの高い行動を誘い込みます。その後、Qで背後に回り込み、相手を危険地帯へ叩き落とします。
- **リソース管理：** 広いマップでは、体力やエネルギーのピックアップ（特定のモードに存在する場合）の場所を把握し、その近くを維持しましょう。
